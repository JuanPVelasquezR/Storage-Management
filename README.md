# Proyecto Final - Gestión de Productos

Full-stack:
- **Backend**: ASP.NET Core + JWT + EF Core (Postgres/Neon)
- **Frontend**: React + TypeScript + Vite + Tailwind (Vercel)

## Variables (resumen)

### Backend (Render)
- `DATABASE_URL`
- `JWT_SECRET`
- `ADMIN_USERNAME` / `ADMIN_PASSWORD`
- `CORS_ORIGINS` (dominio Vercel)

### Frontend (Vercel)
- `VITE_API_BASE_URL` (URL del backend en Render)
